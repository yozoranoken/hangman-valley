# Hangman valley

## Installation
python version: 3.7.4

Install requirements via pip
```sh
pip install -r requirements.txt
```

## Usage
Run the game with:
```sh
python . # inside the project directory
``` 

Edit `db.txt` to manage entries used in the game, where each entry follows the format:
```
> [hint]
> [line 1]
> [line 2]
...
> [line n]
> XXX
```
